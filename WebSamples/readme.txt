1、准备使用报表插件进行WEB报表开发，请先参考帮助中的“报表插件(WEB报表)”部分。 
2、运行WEB报表例子，请参考帮助中“报表插件(WEB报表)->WEB报表例子->准备运行例子”中的说明。 不能直接在 Windows 资源管理器中双击网页文件进行查看，而应该在 web 服务器下或在编程 IDE 下运行。
3、建议先把例子运行起来，最先查看 01.ViewReport 中的例子代码，理解报表插件的运行机理。报表插件嵌入在网页中，报表插件通过属性参数关联报表模板(样式)与报表数据URL，在运行时会根据URL从服务器获取数据，并在客户端电脑上将报表生成并展现出来。也就是说，报表数据来自WEB服务器，报表生成与显示在客户端电脑。
