各个WEB报表例子简要说明 - WEB报表

概要说明

  1. 要运行WEB例子，需要进行一些简单的配置。请打开Grid++Report帮助文档，参考“报表插件(WEB报表)-〉Web报表例子-〉准备运行例子”部分。
  2. 如果在运行过程中出现问题，请参考帮助“报表插件(WEB报表)”部分下的“常见问题解答(FAQ)”与“分析报表不能运行的原因”中的说明。
  3. 初学者请按目录编号顺序学习，首先理解“01.ViewReport”目录中的“DisplayViewReport.htm”与“PrintViewReport.htm”这两个例子，了解报表运行机制请看帮助中的“报表插件(WEB报表)”部分。
  4. Grid++Report快速入门指导: 帮助的“产品介绍->快速入门指导”部分 。

例子根目录下各个文件简介

文件 简要说明
CreateControl.js 报表对象与报表插件创建函数，供报表展现网页调用，后面例子中的报表展现网页都会使用。
GRUtility.js 例子中用到的通用 javascript 函数。
Index.htm 报表例子首页，从此页可以打开运行各个例子。

各目录对应例子简介

目录 简要说明
01.ViewReport 最简单的例子，将报表显示插件嵌入到网页中，以查询显示方式与打印预览方式展现报表，入门应该首先学习这个例子。
02.PrintDirect 报表在网页中不展现，调用报表对象的相关函数，实现报表的打印与数据导出等。
03.Export 通过响应报表导出相关的事件函数，修改导出的选项参数，改变报表的默认导出行为。
04.PrintInForm 实现支票与发票的套打。
05.Picture 提供多种方式为报表中显示图像提供数据。
06.SubReport 实现为子报表提供数据。
07.Chart 实现为图表提供数据。
08.CustomToolbar 自定义报表插件的工具栏按钮，可以删除不需要的按钮，增加自定义按钮。
09.DesignReport 应用报表设计器控件可视化设计报表。
10.SummaryToDetail 实现报表穿透查询，根据汇总报表选中数据打开对应的明细报表，实现按查询条件动态筛选报表数据。
11.BatchData 实现从服务器分批按需获取报表数据，如果报表数据量巨大，可以考虑采用分批按需取数，提高报表加载速度，减轻服务器与网络的负载。
12.NoneDetaiGridData 为没有明细网格的报表提供XML数据。关于WEB报表的XML数据形式：请参考帮助中“报表插件(WEB报表)->报表数据”中的说明。
Advance 进阶例子，实现用代码控制报表的生成与显示，请查看各个例子下的简介文件。
App_Code 仅 aps.net 有此目录，存放报表XML数据生成模块代码
data 提供报表XML数据的网页程序，调用通用方法根据提供的SQL语句产生报表需要的XML数据，为各个例子报表提供数据。
General 参数化使用各个报表插件，实现通用的报表查询显示、报表打印显示、报表设计等功能，主要用在Index.htm中所链接的报表。
grf 例子用到的报表模板文件，即报表样式定义文件。
Misc 一些杂项例子，请查看其中的简介文件。

General目录下各个文件简介

DesignReport.aspx(jsp/php/asp) 通用报表模板设计网页，用报表设计器插件设计报表。
DesignReportSave.aspx(jsp/php/asp) 报表模板设计网页对应的模板保存后台网页程序。
DisplayReport.aspx(jsp/php/asp) 通用报表查询显示插件展现报表网页。
PrintReport.aspx(jsp/php/asp) 通用报表打印显示插件展现报表网页。
ReadReport.aspx(jsp/php/asp) 报表模板读取后台网页程序。
